# generator

A static website generator.

## Getting Started

### Development Installation

```sh
yarn run bootstrap
```

### Running the Tests

```sh
yarn run test
```

### Releasing a New Version

```sh
yarn run version
yarn run build
yarn run pack
yarn run deploy
```
